# Claude Code 벤치마킹 프롬프트

아래 프롬프트를 Claude Code에 그대로 전달한다.

```text
urban-safety-website의 디자인 벤치마킹을 수행해줘.

작업 기준 문서는 다음 파일이다.
- BENCHMARKING_BRIEF.md
- DESIGN_BRIEF.md
- CLAUDE.md

목표:
- 현재 사이트를 시민공익단체/교육기관형 신뢰 웹사이트로 재정렬하기 위한 레퍼런스를 수집하고,
- 실제 코드 반영이 가능한 UI 패턴으로 정리하는 것

중요:
- 예쁜 사이트 링크 모음으로 끝내지 말 것
- 반드시 "우리 사이트 어디에 적용할지"까지 적을 것
- Awwwards는 전체 구조 참고, Dribbble은 부분 컴포넌트 참고로 활용할 것
- 과한 인터랙션/포트폴리오형 사이트는 배제할 것
- 한국어 정보형 사이트에 맞는 가독성과 기관다운 신뢰감이 최우선이다

산출물:
1. 레퍼런스 8~12개
2. 패턴 추출표
3. 홈 화면 개편 방향
4. 공통 UI 개편 방향
5. 바로 구현 가능한 우선순위 5개

파일 출력 형식:
- DESIGN_BENCHMARK_REPORT.md 파일로 저장

추가 요구:
- 홈 페이지 기준으로 hero, trust section, card section, notices list, CTA, footer를 중심 분석할 것
- 모바일 가독성과 한국어 텍스트 밀도 대응도 평가할 것
- 마지막에는 "절대 따라하지 말 것" 섹션을 넣을 것
```

## 추천 후속 흐름

벤치마킹이 끝나면 다음 순서로 이어간다.

1. `DESIGN_BENCHMARK_REPORT.md` 검토
2. 홈 화면 개편 방향 확정
3. 공통 디자인 토큰 정리
4. 실제 코드 반영

# 디자인 브리프 — 대전경실련 도시안전디자인센터

## 클라이언트 정체성

- **기관 성격**: 경실련(경제정의실천시민연합) 산하 법인체
- **포지셔닝**: 공공기관도 민간기업도 아닌 **시민공익단체**
- **핵심 사업**: 교육 + 민간자격 인증 + 사회공헌활동
- **신뢰 기반**: 35년 경실련 운동의 공신력

## 디자인 철학

> "신뢰할 수 있는 이웃 같은 전문가"

- 딱딱한 관공서 ❌
- 화려한 상업 플랫폼 ❌
- 따뜻하지만 전문적인 시민단체 ✅

### 컬러 의미
- **네이비**: 신뢰, 전문성, 경실련의 무게감
- **그린**: 안전, 공익, 시민사회
- **앰버**: 경고/주의(보이스피싱), 행동 촉구

## 타이포그래피

- **본문**: Noto Sans KR (400, 500)
- **제목**: Noto Sans KR (700, 900)
- **강조**: 현재보다 자간 넓게, 행간 여유있게
- **크기 체계**: 14/16/18/24/32/40/48px

## 섹션별 디자인 방향

### 1. 헤더/네비게이션
- 현재: 딱딱한 흰 배경
- 개선: 상단 얇은 컬러 바(그린) + 로고 왼쪽 + 스크롤 시 그림자

### 2. 히어로 섹션
- 현재: 짙은 네이비 그라디언트 (IT스타트업 느낌)
- 개선: 밝은 크림/화이트 배경 + 네이비 텍스트 + 그린 포인트
- 이미지: 시민·교육 활동 느낌의 일러스트 또는 패턴

### 3. 핵심분야 카드
- 현재: 네이비/오렌지 꽉 찬 배경 카드
- 개선: 흰 카드 + 상단 컬러 액센트 라인 + 아이콘 강조

### 4. 자격증 섹션
- 현재: 테두리 카드 (밋밋)
- 개선: 배지/스탬프 느낌 + 신뢰감 있는 인증서 무드

### 5. 전체 톤
- 여백 넉넉하게 (숨쉬는 레이아웃)
- 둥근 모서리 (radius-2xl 기본)
- 그림자 최소화 (flat + subtle)

## 2. 치명적 이슈 (CRITICAL)

### C1. 지도가 실제로 없음
- **위치**: `/about/location`, `/contact`
- **증상**: "지도 영역 (카카오맵/네이버맵 연동)" 플레이스홀더 텍스트가 그대로 노출
- **영향**: 위치 페이지의 **존재 이유가 소멸**. 40-60대 수강생에게 치명적
- **대안**:
  - 단기: Kakao Map JS SDK 1일 만에 연동 (무료, API 키만 발급)
  - 장기: 주변 랜드마크·대중교통 시간대별 실시간 정보 오버레이

### C2. 공지사항이 모두 2024년 더미 데이터
- **위치**: `/notice`
- **증상**: 모든 공지가 `2024-04-01`~`2024-04-25`. 지금은 2026-04-15
- **영향**: "2년 전에 버려진 사이트"로 오인 → **기관 자체의 신뢰 붕괴**
- **대안**:
  - 즉시: 2024년 데이터 전부 삭제 또는 과거 이력으로 라벨링
  - 관리자 CMS(`app/admin/notices`) 이미 구축됨 → 실 공지 3-5건 즉시 작성

### C3. 교육 일정이 2024년인데 "모집 중" 표시
- **위치**: `/education`
- **증상**: `2024-05-10 보이스피싱 5기 개강 / 모집 중` 같은 표시
- **영향**: 수강 의사결정 불가, 허위 고지 우려
- **대안**: `data/schedule.json` 또는 `lib/content-store.ts` 데이터 갱신

### C4. 센터장 사진·이름·직함 없음
- **위치**: `/about/greeting`
- **증상**: 회색 원 플레이스홀더 "센터장 사진" / 서명은 "센터장 **일동**"
- **영향**: 리더십 부재 → 신뢰 구축 실패. 개인 이름·얼굴은 신뢰의 1차 신호
- **대안**: 센터장 1명 선임 프로필(사진·약력 3-5줄·서명) 삽입

### C5. 수강 신청 실제 플로우 부재
- **위치**: `/education/*`, `/certificates/*`
- **증상**: 모든 "자세히 보기" 버튼이 상세 페이지로만 이동. **실제 신청 폼·결제·장바구니 없음**
- **영향**: 상담 전화(`042-254-8060`)에 의존 → 디지털 전환율 0
- **대안**:
  - 1단계: 문의 폼 기반 상담 신청 (현재 구조 재활용)
  - 2단계: 회원가입 + 결제 연동 (토스페이먼츠/카카오페이)

## 3. 주요 이슈 (HIGH)

### H1. 브랜드 팔레트 3중 공존
| 소스 | 메인 컬러 | 상태 |
|---|---|---|
| `BRAND_DESIGN_GUIDELINES.md` | Smart Teal `#00B5AD` | 문서상 권장 |
| `app/globals.css` (로컬 최신) | Navy `#1a3a5c` + Green `#2e7d32` + Amber | 정의됨 |
| 실제 렌더링 | Navy + **Orange 버튼** (`/about`, 모든 CTA) | 미이행 |

- **영향**: 사용자는 일관된 브랜드 경험을 못 받음. 디자인 결정이 방치된 인상
- **대안**: **팔레트 결정부터 우선**. 3개 중 하나 선택 → 전체 토큰 통일 → CTA 포함 전면 치환
- **권장**: 브랜드 가이드의 **Teal**을 받아들이거나, 현재 globals.css의 **Green**을 정식화. Orange는 전면 제거

### H2. `/education/online`이 단순 경유지
- **증상**: 고유 콘텐츠 없음. "자세히 보기"가 자격증 상세로 리다이렉트만
- **영향**: 페이지 존재 가치 없음, SEO 약화
- **대안**: LMS 플랫폼·수강 환경·학습자 후기·샘플 강의 영상 추가. 또는 페이지 통합

### H3. `/about/greeting` 여정 단절
- **증상**: 인사말 읽고 나면 다음 경로 링크 없음 → 사용자 방황
- **대안**: 섹션 하단 CTA 2개 — "운영 중인 자격과정 보기" / "센터 연혁 보기"

### H4. 모바일 히어로 줄바꿈 깨짐
- **위치**: `/` 모바일
- **증상**: "가족을 믿고 맡길 수 있는 / 우리 동네 안전 전문가"가 어색하게 쪼개짐
- **대안**: `break-keep`, `word-break: keep-all`, 적절한 `<br>` 조정

### H5. Sticky Sidebar 모바일 검증 미완료
- **위치**: `/contact`, `/certificates/voice-phishing`
- **조치**: 모바일에서 sidebar는 **상단 또는 하단 고정 CTA 바**로 전환 (스크롤 시 전환율 +30%)
